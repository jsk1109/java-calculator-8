# java-calculator-precourse

<br>

# 기능

1. 사용자가 입력한 문자열에서 쉼표(,) 또는 콜론(:)을 기준으로 숫자를 분리한다.
   
2. 입력값이 "" (빈 문자열)이거나 null일 경우 0을 반환한다.

3. 음수 또는 잘못된 값이 입력될 경우 IllegalArgumentException 예외를 발생시킨다.

4. 분리된 숫자들의 합을 계산한다.

<br>

# 출력

덧셈할 문자열을 입력받아 합계를 출력한다.

입력 : 1,2:3 

출력 : 결과 : 6

입력 : //;\n1;2;3

출력 : 결과 : 6

입력 : 

출력 : 결과 : 0
